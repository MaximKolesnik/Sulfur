To play demo levels, launch the editor, go to File->Open Scene an choose a level.
Play, pause and stop buttons are used to control the simulation.
To turn off debug draw, go to View.
Inspector, the tab on the right, is used to add and modify components
On the bottom Resource tab can be found.
On the left Object tree can be found. Select an object and press Delete to remove the object from the scene
